# Local issue tracker

This folder is the repo's **local-markdown issue tracker** (see
`docs/agents/issue-tracker.md`). The `/to-prd` skill publishes PRDs here as
`docs/issues/<feature-slug>/PRD.md` with a `Status: ready-for-agent` line.

PRDs land in the repo as files — no external tracker required — so the same
artifact a Shaper writes is the one an Orchestrator (or an agent) picks up.
